- [Redux Ducks Pattern](https://github.com/erikras/ducks-modular-redux)

## Instalação

1. Clone este repositórios em sua maquina

2. Instale as dependencias usando:

```
yarn install
```

ou

```
npm install
```

## Executar

```
yarn start
```

ou

```
npm run start
```
